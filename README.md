Projeto para cadastro de produtos <br/>
Consiste em uma tela onde é possivel cadastrar, alterar e remover um produto, passando nome, preço, marca e descrição do mesmo, logo abaixo do formulário é exibido uma tabela com a relação de produtos cadastrados, seguidos de um botão de ação.
Foi utilizado React para desenvolver o front-end, Spring Boot para o back-end,  postgresql como banco de dados, tambem foi utilizado o Docker.
